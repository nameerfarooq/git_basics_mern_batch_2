CSS BOX MODEL

CSS Box Model is the basic foundation and structure of an element or it can also be called the basic layout design of an element on a web page.

The CSS Box Model consists of 4 components:

  • Content - It is the centermost or innermost area of box and contains actual content of an element like Images, Texts, Shapes and other media.
  • Padding - This is the space between the content and the border, Padding can be used to add internal spacing within an element.
  • Border - The Border can be described as the line at the edge of padding, It can provide a visual boundary for the element.
  • Margin - Outermost area of the box is known as Margin, It can be used to provide spacing between one element and the other.
