Introduction to Flexbox

CSS Flexible Box Layout, commonly known as Flexbox, is a CSS3 web layout model that allows you to design complex layouts more efficiently and consistently across different screen sizes.

Why Use Flexbox?

  • Easily create responsive layouts
  • Vertically center content with ease
  • Reorder content without changing HTML structure
  • Control the distribution of space and alignment

Flexbox Terminology

Understanding these terms is essential to working with Flexbox:

Flex Container

The parent element that holds flex items. Created by setting display: flex or display: inline-flex.

Flex Items

The children elements within a flex container.

Main Axis

The primary axis along which flex items are laid out (horizontal by default).

Cross Axis

The axis perpendicular to the main axis (vertical by default).

Flex Container Properties

These properties are applied to the flex container to control the layout of its children:

flex-direction

Defines the direction of the main axis: row, row-reverse, column, or column-reverse.
