1. Introduction

Flexbox (Flexible Box Layout) is a one-dimensional layout system in CSS that allows you to design layouts easily by aligning and distributing space among items in a container.

2. Terminology

Flex Container: The parent element with display: flex.

Flex Items: The direct children of the flex container.

3. Flex Container Properties

4. Flex Item Properties

  • order: Changes order of items.
  • flex-grow: Defines how much an item grows.
  • flex-shrink: Defines how much an item shrinks.
  • flex-basis: Defines initial size of item.
  • flex: Shorthand for grow, shrink, basis.
  • align-self: Overrides alignment for one item.
